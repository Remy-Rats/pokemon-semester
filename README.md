# Pokemon Semester

A Pokémon-themed attendance and class management application built in Godot and NetBeans, using GDScript and Python.

## Features (Planned)

- Create and manage classes
- Assign Pokémon to classes
- Random Pokémon generation
- Pokémon leveling and experience system
- Attendance tracking
- Rival trainer system
- Save and load progress
